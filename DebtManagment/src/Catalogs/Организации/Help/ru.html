Организации

Торговое предприятие может состоять из нескольких организаций, от имени которых будут оформляться документы. В программе ведется учет товародвижения в разрезе организаций торгового предприятия. Для каждой организации определяется список касс каждого магазина (операционных и ККМ), банковских счетов, система налогообложения:общая или упрощенная.

Список организаций открывается по ссылке НСИ – Предприятие – Организации. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные организации, задавая поиск по значениям различных реквизитов.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка. При вводе новой организации следует указать следующие данные:

  • Наименование  в программе - краткое наименование организации, по которому будет удобно находить организацию в списках.

  • Сокращенное, Полное  наименование- сокращенное и полное официальное наименование, которое будет использоваться при печати документов.

  • Юридическое лицо/Индивидуальный предприниматель - указывается кем является организация с юридической точки зрения: юридическим лицом или индивидуальным предпринимателем.

  • Префикс - используется для разделения нумерации документов по разным организациям В начале номера документа добавляется указанный префикс.

  • Система налогообложения – основная система налогообложения организации. Возможен выбор из следующих вариантов: Общая система налогообложения, Упрощенная система налогообложения, Единый налог на вмененный доход, Единый сельскохозяйственный налог, Патентная система налогообложения.

  • Способ учета товаров  –  выбор оценки стоимости товаров - по закупочным или розничным ценам.

  • ИНН, КПП, Код по ОКАТО, Код по ОКПО, ОГРН - информация о кодах организации в соответствии с ее учредительными документами. Используя команду Заполнить ИНН можно автоматически заполнить данные об организации в соответствии с записями об организации в ЕГРЮЛ/ЕГРИП.

  • Обособленное подразделение (филиал). Установка флажка указывает на то, что организация является обособленным подразделением головной организации.  Использование обособленных подразделений (филиалов) позволяет оформлять передачи денежных средств между кассами головной организации и обособленного подразделения или между кассами разных обособленных подразделений.

На закладке Адреса, телефоны вводится информация о контактной информации организации (адреса, телефоны, факс). Для указания адресов можно использовать адресный классификатор. Любой адрес, введенный в программе, можно посмотреть на картах Яндекс.Карты или Google Maps. Для этого в контекстном меню поля ввода адреса (или гиперссылки с адресом) нужно выбрать пункт Адрес на Яндекс.Карты или Адрес на Google Maps. Возможно вводить адреса с абонентским ящиком (а/я), воинской частью (в/ч) и почтовым отделением (п/о). Это позволяет выводить такие адреса, например, на печать в почтовые конверты.

Важно. По умолчанию для организации настроены определённые виды контактной информации: Email, Почтовый адрес, Другая информация, Телефон, Факс, Фактический адрес, Юридический адрес. При необходимости список видов контактной информации можно расширить.

На закладке Ответственные лица вносятся фамилия и инициалы руководителя, бухгалтера и кассира организации для выведения на документы при печати.

Ввод информацию о складах с применением ЕНВД

Чтобы внести информацию о складах, на которых применяется система налогообложения Единый налог на вмененный доход ( ЕНВД), нужно перейти в список Применение систем налогообложения, используя гиперссылку Настроить использование нескольких систем налогообложения и Создать новую запись. В карточке новой записи будет автоматически заполнена организация, из карточки которой создается запись. Дата начала применения системы ЕНВД (Применяется с ) автоматически заполнится первым днем текущего месяца.

Пользователю нужно выбрать магазин и склад, для которых используется ЕНВД. В списке для выбора склада будут выведены торговые залы, для которых данная организация установлена как организация продаж по умолчанию. В новой записи следует выбрать вариант системы налогообложения - Единый налог на вмененный доход.

Если по каким-то причинам действие ЕНВД на данном складе (магазине) отменено, пользователю следует удалить эту запись из списка Применение систем налогообложения.

Изменение системы налогообложения

Для изменения системы налогообложения организации следует нажать на кнопку редактирования рядом с полем Система налогообложения. Откроется список установленных для организации систем налогообложения. Для изменения системы налогообложения следует создать новую запись с помощью кнопки Создать.  В новой записи следует изменить систему налогообложения и установить дату, начиная с которой она начнет действовать (Применяется с). По умолчанию эта  дата устанавливается равной дате начала текущего месяца.
