Договоры

Справочник предназначен для хранения информации обо всех договорах, заключенных между предприятием (собственными организациями) и поставщиками услуг (или платежными агентами - в случае субагентских договоров) по приему платежей от физических лиц.

Список договоров открывается по ссылке Финансы – См. также – Договоры. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов. Дополнительные команды доступны через кнопку Еще.

Новую запись в справочник можно добавить с помощью кнопки Создать (Клавиша Insert). в командной панели списка. Необходимо заполнить поля:

  • Наименование – название договора.
  • Организация - собственное юридическое лицо, заключившее договор.
  • Банковский счет организации - банковский  счет, на котором учитываются денежные средства по договорам контрагентов. Денежные средства платежных агентов и субагентов, как правило, учитываются на отдельном расчетом счете организации.
  • Поставщик услуг - указывается контрагент, который предоставляет услуги. Для поставщика услуг обязательно заполнение информации об ИНН.
  • Признак агента - собственное юридическое лицо, заключившее договор:
    • Агент.
    • Банковский платежный агент.
    • Банковский платежный субагент.
    • Платежный агент.
    • Платежный субагент.
    • Комиссионер.

Для всех агентов и субагентов дополнительно указывается информация о телефонах (платежного агента, оператора по приему платежей и поставщика). Эта информация печатается в качестве реквизитов чека, которые выводятся при печати чека при регистрации оплаты услуг, предоставляемых агентами.

Для банковского платежного агента необходимо заполнить дополнительную информацию, которая также будет распечатана в чеке: наименование, адрес, телефон и ИНН оператора перевода и наименование операции, которую совершает платежный агент.

Для комиссионера заполняется информация о предоставлямой услуге по начислению комиссионного вознаграждения и процент комиссионного вознаграждения. Предварительно информация об этой услуге вводится в справочник Номенклатура.

В соответствии с условиями договора при печати чека при продаже товаров комиссионеру можно выделять НДС или нет при печати в чеке комиссионных товаров.

После заключения договора для него устанавливается статус Действует. Если у договора истек срок действия, то для него устанавливается статус Прекращен.

Для записи элемента справочника используется команда Записать и закрыть (Сочетание клавиш Ctrl + Enter) или Записать (Сочетание клавиш Ctrl + s).
