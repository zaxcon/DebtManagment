Физические лица

Справочник Физические лица содержит список физических лиц, работающих на предприятии (сотрудники и пользователи программы) или являющихся покупателями магазинов торговой сети. В справочнике можно объединять физические лица в произвольные группы и подгруппы, выделяя, например, определенные категории покупателей.

Для физического лица определяются его контактные данные: адрес, телефон, email и т. д. При необходимости пользователь может добавить для физического лица дополнительные виды контактной информации.

Данные о физических лицах, имеющих право оформлять документы и вести хозяйственную деятельность, вносятся в справочник Контрагенты с указанием правового статуса Физическое лицо. В этом случае появляется возможность внести юридические данные физического лица, сведения о банковских счетах и поставляемых товарах.

Перед началом заполнения списка физических лиц необходимо выполнить настройки по ведению списка физических лиц:

· заполнить классификатор документов, чтобы при заполнении данных физического лица пользователь мог выбрать значение из готового списка;

· настроить виды контактной информации для ввода контактной информации по физическим лицам;

· указать список дополнительных свойств (дополнительных реквизитов и сведений).

Список физических лиц открывается по ссылке НСИ – Физические лица - Физические лица). С помощью сервиса Найти, который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

Новый элемент справочника можно ввести с помощью кнопки Создать, и в открывшейся форме заполнить ФИО физического лица и указать контактные данные, а также все требуемые реквизиты согласно внутрикорпоративным дынным.

В панели навигации формы ввода физического лица нужно выбрать пункт Документы (1), далее Создать (2) и в открывшейся форме выбрать из выпадающего списка Вид документа (3). Заполнить необходимые данные в соответствии с удостоверением личности, представленным физическим лицом. Нажать кнопку Записать и закрыть.

Также администратор может указать, в каком магазине работает сотрудник и настроить список смен и выполняемых работ. Эти данные помогут сформировать план выхода сотрудников на неделю (Персонал - Работа магазинов - Планируемое время сотрудников). С помощью команды план работы заполнится в соответствии с указанными для сотрудника сменами и работами.  Команды контекстного меню помогут сформировать план выхода каждого сотрудника, пользователь сможет также внести изменения вручную непосредственно указав количество планируемых рабочих часов.
