Банковский счет контрагента

В справочнике хранится информация о банковских счетах контрагентов (юридических и физических лиц поставщиков и клиентов торгового предприятия).

Банковский счет добавляется для каждого контрагента через его карточку. Для открытия списка банковских счетов контрагента откройте справочник Контрагенты (НСИ -Предприятие -Контрагенты), затем карточку контрагента и выполните команду Перейти -Банковские счета.

С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные счета, задавая поиск по значениям различных реквизитов.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.

В открывшейся форме нового банковского счета введите номер счета и БИК банка. Корсчет, Наименование банка и Город, в котором расположен филиал банка определится автоматически по данным классификатора Банки. Пользователь может выбрать наименование для созданного банковского счета организации из числа автоматически созданных системой или ввести свое.
